# Simple presentations engine

Demo: http://alexvorobyov.ru/presentations/

# Navigation

<table>
    <tr>
        <td>Next slide:</td><td>arrow down, arrow right, page down, space</td>
    </tr>
    <tr>
        <td>Prev slide:</td><td>arrow up, arrow left, page up, backspace</td>
    </tr>
    <tr>
        <td>Last slide:</td><td>end</td>
    </tr>
    <tr>
        <td>First slide:</td><td>home</td>
    </tr>
    <tr>
        <td>Back to list of slides:</td><td>esc</td>
    </tr>
</table>

Also if you move the mouse in the bottom of the screen you can see navigation buttons in overlay in left corner of the screen.

# Browser support

IE8+, Chrome, Firefox, Safari
